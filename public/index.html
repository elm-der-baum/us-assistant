Google Login erforderlich

Dein Google Account ist der Login für diese App. Kalender, Todos, Chat-Verlauf, Safe-Mode-Freigaben und persönliche Einstellungen hängen an diesem Konto.

Google OAuth App konfigurieren

Vor dem Google Login braucht Assistant eine OAuth-Client-ID und ein Client-Secret aus deiner Google Cloud Console.

Google Console Registrierung

  1. Öffne Google Cloud Console → APIs & Services → Credentials.
  2. Aktiviere APIs: Google Calendar API und Google Tasks API.
  3. OAuth Consent Screen einrichten; als Test-User deine Google-Mail hinzufügen.
  4. Create Credentials → OAuth Client ID → Web application.
  5. Authorized JavaScript origins:
    https://findyou.biz
  6. Authorized redirect URIs:
    https://findyou.biz/assistant/oauth/callback
  7. Client-ID und Client-Secret unten eintragen und speichern.
